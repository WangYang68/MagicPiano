## Features

### Multi-Game Support
- Supports key modes for various popular games:
  - Universal 21-key/22-key piano mode
  - Naraka: master mode
  - WhereWindsMeet 36-key
  - Undawn Piano
  - Justice mobile professional version
  - Identity V 36-key
  - LifeAfter Piano 
  - Genshin guitar
  - Heartopia 37
  - Sky 15-key
  - OnceHuman Piano
  - Roblox piano
  - Sizhu Yayun (Traditional Chinese Music)

### Music Management
- Local music import and management
- Cloud music playback
- Recently played records
- My favorites feature
- Music search function
- Paginated music list display

### Playback Control
- Play/Pause/Stop control
- Previous/Next track switching
- Speed control (0.5x-2.0x)
- Progress bar dragging control
- Single track mode (select specific track for playback)
- Track lock/unlock function
- Key signature selection and transposition
- Scheduled playback function

### User Interface
- Multi-language support (Simplified Chinese, Traditional Chinese, English)
- Customizable hotkeys
- Beautiful UI design
- Personal center and user information management
- Check for updates feature

### Follow Play Mode
- Listening mode: Display parsed keys and output
- Follow practice: Display parsed keys, wait for manual key press to continue playback
- Master mode: Display parsed keys for normal playback, but don't output keys

## Usage

### Basic Operations
1. After launching the program, select the game key mode
2. Select the music to play from the music list
3. Click the play button to start performing
4. Adjust playback speed, select tracks, etc. through the bottom control bar

### Hotkey Usage
- Play/Pause: Shift + F9
- Previous track: Shift + F10
- Next track: Shift + F11
- Decrease speed: Shift + Left Arrow
- Increase speed: Shift + Right Arrow
- Favorite: Shift + Insert
- Delete: Shift + Delete

### Advanced Features
1. **Single Track Mode**: Click the "Single Track Mode" button to select the track to play
2. **Track Locking**: Lock a specific track, all playback will use that track
3. **Key Signature Selection**: Click the key signature button to select an appropriate key for transposition
4. **Scheduled Playback**: Set a scheduled playback time, playback will start automatically at that time

# MagicPiano 魔琴

MagicPiano（魔琴）是一款功能丰富的MIDI音乐播放器，专为游戏音乐演奏设计。它支持多种游戏的按键模式，能够将MIDI音乐转换为游戏中的键盘按键，实现自动演奏。

## 功能特点

### 多种游戏支持
- 支持多种热门游戏的按键模式：
  - 通用21键/22键钢琴模式
  - 永劫无间大师模式
  - 燕云36键
  - 黎明觉醒钢琴
  - 逆水寒手游专业版
  - 第五人格36键
  - 明日之后钢琴全键
  - 原神吉他
  - 心动小镇37键
  - 光遇15键
  - 七日世界钢琴
  - Roblox钢琴
  - 丝竹雅韵

### 音乐管理
- 本地音乐导入和管理
- 云音乐播放
- 最近播放记录
- 我的收藏功能
- 音乐搜索功能
- 分页显示音乐列表

### 播放控制
- 播放/暂停/停止控制
- 上一首/下一首切换
- 倍速播放（0.5x-2.0x）
- 进度条拖动控制
- 单音模式（可选择特定音轨播放）
- 轨道锁定/解锁功能
- 调号选择和转调功能
- 定时播放功能

### 用户界面
- 多语言支持（简体中文、繁体中文、英文）
- 热键设置自定义
- 美观的UI设计
- 个人中心和用户信息管理
- 检查更新功能

### 跟弹练习模式
- 试听模式：显示解析的按键并输出
- 跟弹练习：显示解析的按键，等待手动按下相对按键继续播放
- 大师模式：显示解析的按键正常播放，但不输出按键

## 使用方法

### 基本操作
1. 启动程序后，选择游戏按键模式
2. 在音乐列表中选择要播放的音乐
3. 点击播放按钮开始演奏
4. 可通过底部控制栏调整播放速度、选择音轨等

### 热键使用
- 播放/暂停：Shift + F9
- 上一首：Shift + F10
- 下一首：Shift + F11
- 减速：Shift + 左方向键
- 加速：Shift + 右方向键
- 收藏：Shift + Insert
- 删除：Shift + Delete

### 高级功能
1. **单音模式**：点击"单音模式"按钮，选择要播放的音轨
2. **轨道锁定**：锁定特定音轨，所有播放都使用该轨道
3. **调号选择**：点击调号按钮选择合适的调号进行转调
4. **定时播放**：设置定时播放时间，到时间自动开始播放

# MagicPiano 魔琴

MagicPiano（魔琴）是一款功能豐富的MIDI音樂播放器，專為遊戲音樂演奏設計。它支援多種遊戲的按鍵模式，能夠將MIDI音樂轉換為遊戲中的鍵盤按鍵，實現自動演奏。

## 功能特點

### 多種遊戲支援
- 支援多種熱門遊戲的按鍵模式：
  - 通用21鍵/22鍵鋼琴模式
  - 永劫無間大師模式
  - 燕雲36鍵
  - 黎明覺醒鋼琴
  - 逆水寒手遊專業版
  - 第五人格36鍵
  - 明日之後鋼琴全鍵
  - 原神吉他
  - 心動小鎮37鍵
  - 光遇15鍵
  - 七日世界鋼琴
  - Roblox鋼琴
  - 絲竹雅韻

### 音樂管理
- 本地音樂匯入和管理
- 雲音樂播放
- 最近播放記錄
- 我的收藏功能
- 音樂搜尋功能
- 分頁顯示音樂列表

### 播放控制
- 播放/暫停/停止控制
- 上一首/下一首切換
- 倍速播放（0.5x-2.0x）
- 進度條拖動控制
- 單音模式（可選擇特定音軌播放）
- 軌道鎖定/解鎖功能
- 調號選擇和轉調功能
- 定時播放功能

### 使用者介面
- 多語言支援（簡體中文、繁體中文、英文）
- 熱鍵設定自訂
- 美觀的UI設計
- 個人中心和使用者資訊管理
- 檢查更新功能

### 跟彈練習模式
- 試聽模式：顯示解析的按鍵並輸出
- 跟彈練習：顯示解析的按鍵，等待手動按下相對按鍵繼續播放
- 大師模式：顯示解析的按鍵正常播放，但不輸出按鍵

## 系統要求

- Windows 10及以上版本
- 需要以管理員權限執行

## 安裝說明

1. 從發布頁面下載最新版本的安裝包
2. 執行安裝程式並按照提示完成安裝
3. 啟動程式，首次使用需要註冊或登入帳戶

## 使用方法

### 基本操作
1. 啟動程式後，選擇遊戲按鍵模式
2. 在音樂列表中選擇要播放的音樂
3. 點擊播放按鈕開始演奏
4. 可透過底部控制欄調整播放速度、選擇音軌等

### 熱鍵使用
- 播放/暫停：Shift + F9
- 上一首：Shift + F10
- 下一首：Shift + F11
- 減速：Shift + 左方向鍵
- 加速：Shift + 右方向鍵
- 收藏：Shift + Insert
- 刪除：Shift + Delete

### 進階功能
1. **單音模式**：點擊"單音模式"按鈕，選擇要播放的音軌
2. **軌道鎖定**：鎖定特定音軌，所有播放都使用該軌道
3. **調號選擇**：點擊調號按鈕選擇合適的調號進行轉調
4. **定時播放**：設定定時播放時間，到時間自動開始播放
